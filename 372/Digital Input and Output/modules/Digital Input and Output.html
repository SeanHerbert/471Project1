Digital Input and Output

Module Objectives

If you successfully complete this module, you will be able to:

  1. (Course Objective A) Memorize the purpose for PORT, PIN, and DDR special function registers in digital I/O
  2. (Course Objective A) Construct code for manipulating and retrieving single bits using SFR macros based on provided register maps
  3. (Course Objective B) Using digital I/O, develop a program to manipulate several LEDs and a switch
